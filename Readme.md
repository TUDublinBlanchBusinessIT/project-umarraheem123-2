# CapTracker

CapTracker is a mobile application designed to monitor gym capacity in real time.  
It records the number of people entering and exiting the gym and displays the current occupancy to users and staff.  
The app supports both Android and iOS and is built using React Native with Firebase backend services.

## Features
- Track number of people entering and leaving the gym
- Real-time capacity updates
- Firebase backend for data storage and syncing
- Cross-platform support (Android & iOS)

## Tech Stack
- React Native
- Expo Go
- Firebase (Firestore / Authentication)
- JavaScript / TypeScript
